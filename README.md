# Mini Social Media App

A MERN Stack mini social media application built for the CodeAlpha Internship.

## Features

- User Registration
- User Login
- Create Posts
- Upload Images
- Like Posts
- Comment on Posts
- Delete Own Posts
- User Profiles

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Axios
- Framer Motion

## Installation

Frontend

npm install
npm run dev

Backend

npm install
node server.js
